构建下一代 AI 助手

OpenClaw 是一个开源、可扩展的 AI 助手平台，帮助开发者构建自主智能体，集成多种工具和服务，实现真正的自动化工作流。

强大特性

多工具集成

内置文件读写、Shell 执行、网页搜索、浏览器控制、飞书文档、Git 等工具，轻松扩展新技能。

自主智能体

支持创建分层智能体团队，领导与员工协作，实现复杂任务分解与自动执行。

安全可控

严格的权限控制与隐私保护，所有操作均在用户环境中进行，确保数据不泄露。

工作原理

1. 安装与配置

通过 npm/pnpm 一键安装 OpenClaw，配置您的模型 API 密钥和工具权限，几分钟内即可启动。

2. 技能扩展

从技能商店安装现有技能，或使用技能创建器编写自定义技能，赋予智能体新的能力。

3. 自动化工作流

通过自然语言指令驱动智能体执行复杂任务，如文件整理、信息收集、定时监控、网页部署等。

快速开始

只需几步，即可拥有您的专属 AI 助手。


# 使用 pnpm 全局安装
pnpm add -g openclaw

# 初始化配置
openclaw init

# 启动网关服务
openclaw gateway start

# 开始与智能体对话
openclaw chat
